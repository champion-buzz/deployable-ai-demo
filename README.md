# Deployable AI Demo

This is a simple static demo page for explaining my AI projects.

It is built with plain HTML, CSS, and JavaScript so it can be deployed easily on GitHub Pages, Netlify, or any static host.

## What The Demo Does

The page has a small question box. It gives sample answers for:

- RAG
- tool-using agents
- evals

It is not connected to a real backend.

## Tech Used

- HTML
- CSS
- JavaScript

## Project Files

```text
index.html      Demo page
style.css       Styling
script.js       Small answer logic
README.md       Project notes
```

## Run It Locally

Open:

```text
index.html
```

Or run:

```bash
python -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Deploy With GitHub Pages

This demo is deployed here:

```text
https://champion-buzz.github.io/deployable-ai-demo/
```

One simple way to deploy a similar static project:

1. Push this project to GitHub.
2. Go to repo settings.
3. Open `Pages`.
4. Choose the main branch.
5. Save and wait for GitHub to publish the link.
